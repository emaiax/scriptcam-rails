# Scriptcam::Rails

ScriptCam, tastefully bundled for the Rails 3.1 asset pipeline. Sweet!

## Installation

Add this line to your application's Gemfile:

    gem 'scriptcam-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install scriptcam-rails
